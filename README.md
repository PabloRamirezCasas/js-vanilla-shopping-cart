# 🛒 Carrito de Compras - JS Vanilla (Ruta MERN)

![Estado del Proyecto](https://img.shields.io/badge/Estado-En%20Desarrollo-green)

Este proyecto es una aplicación funcional de carrito de compras donde se aplican conceptos fundamentales de JavaScript Moderno antes de saltar a frameworks como React.

## 🚀 Funcionalidades
- **Gestión de Productos:** Agregar y eliminar cursos del carrito de forma dinámica.
- **Persistencia:** (Opcional si lo harás luego) Uso de LocalStorage para mantener el carrito tras recargar.
- **Validaciones:** Control de duplicados en la selección de cursos.
- **UI Dinámica:** Actualización del DOM en tiempo real sin recargar la página.

## 🛠️ Tecnologías y Conceptos Aplicados
- **JavaScript ES6+:** Arrow functions, Destructuring, Template Strings.
- **DOM Manipulation:** `querySelector`, `addEventListener`, delegación de eventos (`event delegation`).
- **CSS Framework:** Skeleton (para un diseño limpio y rápido).
- **Control de Versiones:** Git & GitHub bajo el estándar de *Conventional Commits*.

## 💡 Lo que aprendí en este proyecto
> "Aquí es donde demuestras tu madurez intelectual". 
En este ejercicio profundicé en la **Delegación de Eventos**, entendiendo por qué es más eficiente asignar un solo listener al contenedor padre en lugar de múltiples listeners a cada botón de 'Agregar'.
